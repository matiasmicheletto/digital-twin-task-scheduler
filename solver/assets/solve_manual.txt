NAME  
   solve - Computes the optimal allocation of tasks to network nodes

SYNOPSIS  
   solve -t [TASK_FILE] -n [NETWORK_FILE] -o [OUTPUT_FORMAT]

DESCRIPTION:  
   

OPTIONS:  
   -h, --help     Display this help message.  
   -t, --tasks    File with the tasks the instance. Must be in json format.
   -n, --network  File with the network of the instance. Must be in json format.
   -o, --output   (optional) Output format. Must be "json" or "text". Default value is "text".  

EXAMPLES:  
   solve -t tasks.json -n network.json -o json
