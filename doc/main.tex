\section{Candidate representation}
A candidate $c$ is represented as a tuple of two arrays:
\begin{itemize}
	\item \texttt{server\_indices[i]}: server assigned to task $i$.
	\item \texttt{priorities[i]} $\in [0,1]$: priority of task $i$ in the server assigned to it.
\end{itemize}
